Gets the maxresdefault thumbnail for YouTube videos from original video URL. Not associated with YouTube or Google/Alphabet.

thumbsget4mac is a port of thumbsget to macOS using Visual Studio 2019's "Cocoa App" C# template. You can get the code for thumbsget.
